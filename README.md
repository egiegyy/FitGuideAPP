# FitGuide

FitGuide is a Flutter-based fitness application designed to help users manage and track their workout routines efficiently.

---

## Features

- Create and manage workout routines
- Track workout progress (weight and reps)
- Progress visualization using charts
- Clean and modern user interface
- Smooth navigation experience

---

## Upcoming Features

The application is under active development. Planned improvements include:

- AI-based chatbot for personalized workout recommendations based on user needs
- Dark mode and light mode support
- UI and asset improvements
- Additional exercises and equipment variations
- Performance optimizations
